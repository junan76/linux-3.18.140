Name

drm_helper_mode_fill_fb_struct — fill out framebuffer metadata

 struct drm_mode_fb_cmd2 * mode_cmd);
 

Arguments

fb

Description

This helper can be used in a drivers fb_create callback to pre-fill the fb's metadata fields.
