Name

intel_sanitize_enable_execlists — sanitize i915.enable_execlists

Synopsis

int intel_sanitize_enable_execlists (struct drm_device * dev,
 int enable_execlists);
 

Arguments

dev

DRM device.

enable_execlists

value of i915.enable_execlists module parameter.

Description

Only certain platforms support Execlists (the prerequisites being support for Logical Ring Contexts and Aliasing PPGTT or better), and only when enabled via module parameter.

Return

1 if Execlists is supported and has to be enabled.
