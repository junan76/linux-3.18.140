Name

__unqueue_futex — Remove the futex_q from its futex_hash_bucket

Synopsis

void __unqueue_futex (struct futex_q * q);
 

Arguments

q

The futex_q to unqueue

Description

The q->lock_ptr must not be NULL and must be held by the caller.
