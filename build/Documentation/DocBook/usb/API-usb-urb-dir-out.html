Name

usb_urb_dir_out — check if an URB describes an OUT transfer

Synopsis

int usb_urb_dir_out (struct urb * urb);
 

Arguments

urb

URB to be checked

Return

1 if urb describes an OUT transfer (host-to-device), otherwise 0.
