Name

usb_urb_dir_in — check if an URB describes an IN transfer

Synopsis

int usb_urb_dir_in (struct urb * urb);
 

Arguments

urb

URB to be checked

Return

1 if urb describes an IN transfer (device-to-host), otherwise 0.
