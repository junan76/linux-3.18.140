Prev Chapter 9. Serial Peripheral Interface (SPI) Next

Name

spi_write — SPI synchronous write

Synopsis

int spi_write (struct spi_device * spi,
 const void * buf,
 size_t len);
 

Arguments

spi

device to which data will be written

buf

data buffer

len

data buffer size

Context

can sleep

Description

This writes the buffer and returns zero or a negative error code. Callable only from contexts that can sleep.
