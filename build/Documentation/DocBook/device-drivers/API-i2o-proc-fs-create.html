Name

i2o_proc_fs_create — Create the i2o proc fs.

Synopsis

int i2o_proc_fs_create ( void);
 

Arguments

void

no arguments

Description

Iterate over each I2O controller and create the entries for it.

Returns 0 on success or negative error code on failure.
