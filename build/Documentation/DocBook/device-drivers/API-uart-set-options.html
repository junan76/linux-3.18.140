Name

uart_set_options — setup the serial console parameters

Synopsis

int uart_set_options (struct uart_port * port,
 struct console * co,
 int baud,
 int parity,
 int bits,
 int flow);
 

Arguments

port

pointer to the serial ports uart_port structure

co

console pointer

baud

baud rate

parity

parity character - 'n' (none), 'o' (odd), 'e' (even)

bits

number of data bits

flow

flow control character - 'r' (rts)
