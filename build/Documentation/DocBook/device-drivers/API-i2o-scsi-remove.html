Name

i2o_scsi_remove — Remove I2O device from SCSI core

Synopsis

int i2o_scsi_remove (struct device * dev);
 

Arguments

dev

device which should be removed

Description

Removes the I2O device from the SCSI core again.

Returns 0 on success.
