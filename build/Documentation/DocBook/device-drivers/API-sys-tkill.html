Name

sys_tkill — send signal to one specific task

Synopsis

long sys_tkill (pid_t pid,
 int sig);
 

Arguments

pid

the PID of the task

sig

signal to be sent

Description

Send a signal to only one task, even if it's a CLONE_THREAD task.
