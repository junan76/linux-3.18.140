Name

finish_wait — clean up after waiting in a queue

Synopsis

void finish_wait (wait_queue_head_t * q,
 wait_queue_t * wait);
 

Arguments

q

waitqueue waited on

wait

wait descriptor

Description

Sets current thread back to running state and removes the wait descriptor from the given waitqueue if still queued.
