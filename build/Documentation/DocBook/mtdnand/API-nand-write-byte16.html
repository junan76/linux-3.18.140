Name

nand_write_byte16 — [DEFAULT] write single byte to a chip with width 16

Synopsis

void nand_write_byte16 (struct mtd_info * mtd,
 uint8_t byte);
 

Arguments

mtd

MTD device structure

byte

value to write

Description

Default function to write a byte to I/O[7:0] on a 16-bit wide chip.
