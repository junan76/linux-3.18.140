Name

sas_phy_delete — remove SAS PHY

Synopsis

void sas_phy_delete (struct sas_phy * phy);
 

Arguments

phy

SAS PHY to remove

Description

Removes the specified SAS PHY. If the SAS PHY has an associated remote PHY it is removed before.
