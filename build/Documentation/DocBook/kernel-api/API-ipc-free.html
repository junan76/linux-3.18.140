Prev IPC utilities Next

Name

ipc_free — free ipc space

Synopsis

void ipc_free (void * ptr,
 int size);
 

Arguments

ptr

pointer returned by ipc_alloc

size

size of block

Description

Free a block created with ipc_alloc. The caller must know the size used in the allocation call.
