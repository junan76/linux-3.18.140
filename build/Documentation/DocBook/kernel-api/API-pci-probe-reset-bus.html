Name

pci_probe_reset_bus — probe whether a PCI bus can be reset

Synopsis

int pci_probe_reset_bus (struct pci_bus * bus);
 

Arguments

bus

PCI bus to probe

Description

Return 0 if bus can be reset, negative if a bus reset is not supported.
