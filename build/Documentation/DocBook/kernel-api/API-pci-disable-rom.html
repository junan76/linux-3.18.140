Name

pci_disable_rom — disable ROM decoding for a PCI device

Synopsis

void pci_disable_rom (struct pci_dev * pdev);
 

Arguments

pdev

PCI device to disable

Description

Disable ROM decoding on a PCI device by turning off the last bit in the ROM BAR.
