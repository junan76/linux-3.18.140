Name

pci_probe_reset_slot — probe whether a PCI slot can be reset

Synopsis

int pci_probe_reset_slot (struct pci_slot * slot);
 

Arguments

slot

PCI slot to probe

Description

Return 0 if slot can be reset, negative if a slot reset is not supported.
