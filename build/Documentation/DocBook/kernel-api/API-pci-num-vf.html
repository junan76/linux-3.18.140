Name

pci_num_vf — return number of VFs associated with a PF device_release_driver

Synopsis

int pci_num_vf (struct pci_dev * dev);
 

Arguments

dev

the PCI device

Description

Returns number of VFs, or 0 if SR-IOV is not enabled.
