Prev IPC utilities Next

Name

ipc_obtain_object —

Synopsis

struct kern_ipc_perm * ipc_obtain_object (struct ipc_ids * ids,
 int id);
 

Arguments

ids

ipc identifier set

id

ipc id to look for

Description

Look for an id in the ipc ids idr and return associated ipc object.

Call inside the RCU critical section. The ipc object is *not* locked on exit.
