Name

bio_unmap_user — unmap a bio

Synopsis

void bio_unmap_user (struct bio * bio);
 

Arguments

bio

the bio being unmapped

Description

Unmap a bio previously mapped by bio_map_user. Must be called with a process context.

bio_unmap_user may sleep.
