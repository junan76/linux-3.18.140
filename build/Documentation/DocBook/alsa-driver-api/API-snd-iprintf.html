Name

snd_iprintf — printf on the procfs buffer

Synopsis

int snd_iprintf (struct snd_info_buffer * buffer,
 const char * fmt,
  ...);
 

Arguments

buffer

the procfs buffer

fmt

the printf format

...

variable arguments

Description

Outputs the string on the procfs buffer just like printf.

Return

The size of output string, or a negative error code.
