Prev AC97 Codec API Next

Name

snd_ac97_update_bits — update the bits on the given register

Synopsis

int snd_ac97_update_bits (struct snd_ac97 * ac97,
 unsigned short reg,
 unsigned short mask,
 unsigned short value);
 

Arguments

ac97

the ac97 instance

reg

the register to change

mask

the bit-mask to change

value

the value to set

Description

Updates the masked-bits on the given register only when the value is changed.

Return

1 if the bits are changed, 0 if no change, or a negative code on failure.
