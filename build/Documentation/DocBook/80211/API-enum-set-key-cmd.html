Name

enum set_key_cmd — key command

Synopsis

enum set_key_cmd {
  SET_KEY,
  DISABLE_KEY
};

Constants

SET_KEY

a key is set

DISABLE_KEY

a key must be disabled

Description

Used with the set_key callback in struct ieee80211_ops, this indicates whether a key is being removed or added.
