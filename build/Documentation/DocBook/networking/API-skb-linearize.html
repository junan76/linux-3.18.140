Name

skb_linearize — convert paged skb to linear one

Synopsis

int skb_linearize (struct sk_buff * skb);
 

Arguments

skb

buffer to linarize

Description

If there is no free memory -ENOMEM is returned, otherwise zero is returned and the old skb data released.
