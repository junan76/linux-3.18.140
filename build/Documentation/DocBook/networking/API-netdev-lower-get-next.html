Name

netdev_lower_get_next — Get the next device from the lower neighbour list

Synopsis

void * netdev_lower_get_next (struct net_device * dev,
 struct list_head ** iter);
 

Arguments

dev

device

iter

list_head ** of the current position

Description

Gets the next netdev_adjacent from the dev's lower neighbour list, starting from iter position. The caller must hold RTNL lock or its own locking that guarantees that the neighbour lower list will remain unchainged.
