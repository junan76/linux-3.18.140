Name

rio_release_inb_pwrite — release inbound port-write message service

Synopsis

int rio_release_inb_pwrite (struct rio_dev * rdev);
 

Arguments

Description

Removes callback from the rio_dev structure. Returns 0 if the request has been satisfied.
