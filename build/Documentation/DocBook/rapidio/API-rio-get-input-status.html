Synopsis

int rio_get_input_status (struct rio_dev * rdev,
 int pnum,
 u32 * lnkresp);
 

Arguments

rdev

RIO devive to issue Input-status command

pnum

Device port number to issue the command

lnkresp

Response from a link partner
