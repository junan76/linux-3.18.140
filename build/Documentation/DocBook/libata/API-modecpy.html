Name

modecpy — Prepare response for MODE SENSE

Synopsis

void modecpy (u8 * dest,
 const u8 * src,
 int n,
 bool changeable);
 

Arguments

dest

output buffer

src

data being copied

n

length of mode page

changeable

whether changeable parameters are requested

Description

Generate a generic MODE SENSE page for either current or changeable parameters.

LOCKING

None.
